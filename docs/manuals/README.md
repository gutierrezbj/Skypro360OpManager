## Cuándo actualizar

Cada vez que se libere una funcionalidad que afecte al flujo de usuario:
- Nueva pestaña en el sidebar
- Cambio de comportamiento en formularios
- Nuevos roles o permisos
- Cambios en formularios AESA

Pull requests bienvenidos.
